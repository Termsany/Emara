Product description:
EMARA | عمارة is a SaaS platform for interior architecture, interior design, fit-out, and execution project management. It helps architecture and interior design offices manage the full workflow from client lead to design approval, BOQ, payments, and handover.

Main workflow:
Lead / Client → Quotation by square meter → Contracted Project → Site Survey → 2D Plan → Mood Board → 3D Design → Client Approval / Revision → Shop Drawings → Basic BOQ → Payments → Client Portal → Project Dashboard
